# Distributed MongoDB UI

This project is a **distributed MongoDB system** with a **responsive web dashboard**.  

## Features

- Insert, Get, Update, Delete users
- Login page and dashboard UI
- Primary + replica node handling in MongoDB
- Responsive, modern design with buttons and colors

## Tech Stack

- Python (Flask)
- MongoDB Atlas (cloud)
- HTML, CSS, JS
- Gunicorn for deployment
- Render for hosting
